# Doctortracker

 [![API](https://img.shields.io/badge/API-15%2B-brightgreen.svg?style=flat)](https://android-arsenal.com/api?level=15)

Aplikasi mobile untuk android yang dimana fungsi khususnya untuk mencari
dokter praktik terdekat yang ada di kota Makassar

<p align="center"><img src="https://github.com/rahsyarigami/doctortracker/blob/master/s01.PNG" width="100%" /></p>



## Fitur
> - Pencarian lokasi dokter praktik terdekat
> - Pasien mendaftar secara online
> - Melihat peta lokasi dokter yang ada di Makassar

<p align="center"><img src="https://github.com/rahsyarigami/doctortracker/blob/master/s02.PNG" width="100%" /></p>
